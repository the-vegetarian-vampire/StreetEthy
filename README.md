## To Run

1. Install Dependencies:   
$ npm install

2. Run tests   
$ npx hardhat test

3. Start Hardhat node   
$ npx hardhat node

4. Run deployment script   
In a separate terminal execute: $ npx hardhat run ./scripts/deploy.js --network localhost

5. Start frontend   
$ npm run start
